# 🤝 RelationAI - 人間関係・学習サポートアプリ

大学生の対人関係・学習の悩みをAIがサポートするWebアプリです。

## 🚀 ローカルで動かす

```bash
npm install
npm run dev
```

## ☁️ Vercelにデプロイ

1. GitHubにpush
2. [vercel.com](https://vercel.com) でNew Project → リポジトリを選択
3. デフォルト設定のままDeploy！

## 機能

| タブ | 内容 |
|------|------|
| 🤝 AI相談 | 対人関係の悩みをAIに自由相談 |
| ✍️ 頼み方生成 | 相手・状況に応じたフレーズを3パターン自動生成 |
| 🗺️ 関係マップ | 人間関係を視覚的に整理 ＋ AIアドバイス |
| 📚 学習相談 | 勉強の進め方・計画・暗記法などを相談 |
